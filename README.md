# school-erp
A Complete School ERP Software Built With Vue JS.

Env

```bash
  cp .env.example .env

```
Key Genrate

```bash
  php artisan key:generate
```

## Run Locally



Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
Laravel

```bash
  composer install
```

Start the server

```bash
  php artisan serve
```
